MKTJS Space (NES Edition) - Offline package
-------------------------------------------
Files included:
- index.html        : The game. Open this file in a modern browser to play.
- shoot.wav         : Local shoot sound (simple synthetic blip).
- bgmusic.wav       : Local background chiptune-like loop.
- PressStart2P.woff2: Placeholder font file. Replace with real PressStart2P.woff2 for authentic NES font.

Notes:
- To replace the font, download 'PressStart2P.woff2' and overwrite the placeholder in this folder.
- For best pixel-perfect scaling, open index.html and play in a browser on desktop or mobile.
- You can upload this folder to GitHub Pages, Netlify, or Vercel to host the game publicly.

Enjoy!
